Binding

Binding simple

Le binding simple permet d'intégrer à l'affichage une expression typescript dont la valeur sera évaluée.

Par exemple avec la variable nom dans le .ts: {{ nom }} !

Utilisation dans un attribut

Binding attribut

Dans le cas ou toute la valeur d'un attribut peut être donnée via une expression typescript, on peut utiliser le binding d'attribut en mettant l'attribut ciblé entre [] (ex: [class]=" ... ", )

Utilisation dans un attribut

Two-Way Binding

Il est possible qu'on veuille récupérer des infos de l'utilisateurs et l'utiliser dans le .ts. Dans ce cas nous devons lier un input à une variable via ngModel.

Par exemple, récupérons le nom de l'utilisateur (affiché plus haut).

Event binding

Pour réagir aux événements, une autre manière de procéder est mise en place par rapport au JS. Pour réagir à un événement, on met cet event entre parenthèse est on donne l'exécution en valeur tel un attribut. ex: (click)="onClick()"
La fonction liée doit être définie dans le .ts.

Les events charient souvent des infos en se déclenchant. On peut récupérer ces infos grâce à $event.
