Les directives

Les directives de composants

NgClass

ngClass applique une classe css si un condition est vrai

NgStyle

Permet de donner une valeur 'typescript' à une propriété css de la balise

Les directives structurelles

*ngIf

Le *ngIf permettra d'intégrer ou pas une balise dans le code.

*ngFor

*ngFor permet de répeter du code pour chaque valeur dans une liste

  • index: {{i}} - {{nbr}}
  • pair

[ngSwitch], *ngSwitchCase et *ngSwitchDefault

ngSwitch permet d'afficher un de plusieurs bouts de codes selon la valeur d'une expression 'typescript'

la couleur est rouge

la couleur est vert

la couleur est bleu

autre couleur

Les directives customs

Il est possible de créer ses propres directives.

Il en ressort qu'une directive peut être considéré comme un attribut custom qui affectera le comportement de la balise (custom ou pas) sur laquelle il sera apposé.
