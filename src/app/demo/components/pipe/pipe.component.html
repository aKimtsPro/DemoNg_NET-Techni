Les pipes

Les pipes servent à une transformation d'information directement dans le template.

Pipes prédéfinis

Par exemple, je souhaite afficher une date: {{ date | date:'dd MMMM yy' | uppercase }}

Pipes custom

On peut aussi créer nos propres pipes.

Ecriture d'un animal sans pipe: {{ animal.nom }} (espece: {{animal.espece}}, age: {{animal.age}} ans)

Ecriture d'un animal sans pipe: {{ animal | animal }}

Note supplémentaire
