Les pipes

Les pipes servent à une transformation d'information directement dans le template.

Pipes prédéfinis

Par exemple, je souhaite afficher une date: {{ date | date:'dd MMMM yy' | uppercase }}

Pipes custom

On peut aussi créer nos propres pipes.

Ecriture d'un animal sans pipe: {{ animal.nom }} (espece: {{animal.espece}}, age: {{animal.age}} ans)

Ecriture d'un animal avec pipe custom: {{ animal | animal }}

Note supplémentaire

Il est possible de donner un paramètre après le premier sans donner le premier.

Attention lorsqu'on pipe des objects

Les pipes 'pures' ne détectent pas les modif à l'intérieur des objects sur lesquels ils sont appliqués. Les rendre impures peut être une solution mais attention au problèmes de perf. que cela peut entraîner
