Bienvenu dans la section des exercices

Cette section a pour but de présenter les différentes corrections aux exercices proposés:

exo 1: les components et le binding

L'objectif est d'avoir un select permettant de choisir la couleur d'un carré parmi rouge, vert ou bleu

exo 2: les pipes

L'objectif est de pouvoir transformer un donnée en une autre pour facilité l'affichage. Dans ce cas, des secondes en une chaine de caractère contenant minutes et secondes converties

exo 3.1: les directives

L'objectif est d'utiliser les directives de composants et les directives structurelles pour arriver au comportement souhaité.

exo 3.2: Les directives customs

L'objectif est de créer une directive custom et l'utiliser.

exo 4: les Input/Output

L'objectif est de manier les Input et Output dans le but faire passer les infos de parent à enfant et inversément.
