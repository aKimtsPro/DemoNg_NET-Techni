Import/Export

Pour utiliser dans un élément déclaré dans un module un élément déclaré dans un autre, il faut:

  • exporter depuis le module qui le déclare l'élément que l'on veut utiliser
  • importer le module qui export l'élément dans le module qui déclare l'élément qui veut l'utiliser

Par exemple le module Panier déclare un PanierComponent. Dans le AppComponent de mon module principal, j'aimerais placer mon panier. Je fais alors les actions suivantes:

  • Dans PanierModule, je rajoute le PanierComponent (qu'on retrouve dans les déclarations) au exports (rajouter exports au même niveau que imports si nécessaire)
  • Dans AppModule, je rajoute PanierModule dans les imports
